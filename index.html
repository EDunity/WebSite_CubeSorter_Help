[ 前書き ]
「CubePop」は僕が人生で初めて作ったアプリです。 このゲームは、ゲーム開始時に個人のアカウントを作成するので、いつでもインターネット環境さえあればゲームモード別の世界ランキングで自分の順位を確認することができます。 これからも日々改良していく予定です。



[ 作者について ]
Unityインターハイ2019でゴールドアワードを受賞した高校3年生です。 毎日Unityを触っているので「毎日Unity」という名前でやっています。 モデリングと シナリオを考えるのが少し苦手です。でもゲーム開発はとても好きなので、頑張って苦手を克服したいです。

( ゲームの始め方 )

[ アカウントを持っている場合 ]
まずSignInを押してSignIn画面に移動してください。 次に以前作成したPlayerNameとPassWordを入力し、SignInを押してSignInを完了してください。 入力したPlayerNameまたはPassWordが間違っているとSignInを完了することが出来ません。 SignInが完了するタイトル画面に移動します。 タイトル画面に移動したらGameStartを押してゲームを開始してください。
※インターネット環境がないとSignInはできません。


[ アカウントを持っていない場合 ]
まずSignUpを押してSignUp画面に移動してください。 次にPlayerNameとPassWordを決め、入力しSignUpを押してSignUpを完了してください。 入力したPlayerNameが既に他のプレイヤーに使用されていた場合、SignUpを完了することはできません。 SignUpが完了するとSignIn画面に移動します。 SignIn画面では、先ほどSignUp画面で入力したPlayerNameとPassWordを入力し、SignInを押してSignInを完了してください。 SignInが完了するタイトル画面に移動します。 タイトル画面に移動したらGameStartを押してゲームを開始してください。
※インターネット環境がないとSignUp/SignInはできません。

( ゲームの遊び方 )



キューブを消去して、スコアを稼ぐだるま落としのようなゲームです。 積みあがっているキューブの最下部にあるキューブの色と同じ色のボタンを押すことでキューブを消去できます。 連続してキューブを消去するとコンボが発生し、ボーナススコアを得られます。 どのゲームモードも制限時間は45秒です。制限時間内に高いスコアを出して世界ランキング上位を目指しましょう！

( ゲームモード一覧 )



[ 1Color ]
キューブの色が1色しかないゲームモードです。 3つあるゲームモードの中では最も簡単なゲームモードですが、ゲームオーバー時にもらえる経験値が少ないです。



[ 2Color ]
キューブの色が2色のゲームモードです。 難易度が普通で1番人気のあるゲームモードです。 ゲームオーバー時にもらえる経験値は、多くも少なくもないです。



[ 3Color ]
キューブの色が3色のゲームモードです。 3つのゲームモードの中では最も難しいゲームモードですが、ゲームオーバー時に最も多く経験値をもらうことが出来ます。

( Rankingについて )



計4つの世界ランキングを用意しました。 ゲームモード別のランキングとプレイヤーのレベルのランキングです。 ゲームモード別のランキングでは、全プレイヤーの各ゲームモードでの最高スコアをスコアの高い順に表示します。 プレイヤーのレベルのランキングでは、全プレイヤーのプレイヤーレベルの順位をレベルの高い順に表示します。
※インターネット環境がないとランキングは見れません。

( Customizeについて )



[ CubeSkin ]
キューブの模様を変更します。
6種類の中から1つを選んでください。
※プレイヤーのレベルが30以上ないとキューブの模様を変更できません。


[ CubeColor ]
キューブの色を変更します。
8種類の中から3つを選んでください。
※プレイヤーのレベルが40以上ないとキューブの色を変更できません。
